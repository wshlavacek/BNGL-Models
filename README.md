# BNGL-Models

A collection of BioNetGen Language (BNGL) models drawn from the published literature.

## Models

| Model | Reference |
|-------|-----------|
| Bivalent ligand-receptor cyclic dimers | Posner et al., 1995 |
| Cooperative bivalent binding | Posner et al., 2004 |
| Genetic toggle switch | Gardner et al., 2000 |
| Heterogeneous bivalent hapten-receptor crosslinking | Goldstein et al., 1980 |
| IgE antigen pulse binding | Harmon et al., 2017 |
| Kinetic proofreading | Hlavacek et al., 2001 |
| Lac operon lactose induction | Dreisigmeyer et al., 2008 |
| Steric effects in ligand-receptor binding | Hlavacek et al., 1999 |
| Symmetric bivalent hapten-receptor crosslinking | Dembo & Goldstein, 1978 |
| TLBR aggregation | Yang et al., 2008 |
| Trivalent ligand-bivalent receptor aggregation | Macken & Perelson, 1982 |
